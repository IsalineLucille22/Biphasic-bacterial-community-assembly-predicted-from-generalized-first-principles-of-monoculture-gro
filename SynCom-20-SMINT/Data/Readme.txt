README

This folder contains the following input and output files:

1) MergedData.xlsx 

We use sheet 8 'Parameter sets', with the yields and sorted names
We use sheet 9 'Summary parameters Senka data' with the maximum growth rates and lag times

2) S20_S21_abs_abund_cfu_Senka.xlsx 

We use the SynCom-20 data in sheet 4 'Mix SynCom 20 new + old (2)' for the SAO-algorithm  (the test set alone is provided as copy in the sheet 'SynCom 20 g per ml')

3) The output of the growth simulations as .mat files

Matlab files:

6)data_to_save_Inter_v2.m the output of the SAO algorithm saved in the .mat file
7)Zenodo_SynCom20_Paperz.m the output file name for uploading to GitHub/Zenodo

The folder Abundances Data/ contains the files with the different experimental data:

8)Microbacterium_r2a50xdil&ecoplate.xlsx 
Monoculture growth of Microbacterium - repeated

9)PTYG_OD600_21strain_Senka&Tania.xlsx 
Monoculture growth on PTYG

10)S20_S21_abs_abund_cfu_Senka.xlsx 
File containing all soil-grown data in absolute abundances (SynCom20, SynCom21, SynCom21MT, monocultures grown in soil, weekly dilution experiment)

11)Senka_Biolog_PlateReader_allstrainscombined.xlsx 
Biolog plate growth of Monocultures

12)Senka_OD600_succinategrowth_pooleddata.xlsx 
Monoculture growth on Succinate
