README

This folder has four subfolders.

Data/
The experimental datasets used for the training and validation of the SMINT-parameters by the simulated-annealing optimization (SAO) algorithm

has a subfolder /Abundances Data/ that has the monoculture growth curves used in /Fitting/ 

Figures/
To collect the output from running the Community growth model.

Fitting/
The R scripts that fit Monod growth parameters from monoculture experiments (succinate, PTYG, Biolog plates) that is used as input for scripts in /Scripts/

Scripts/
The various MATLAB scripts and subroutines for the growth demography simulations and the SAO
