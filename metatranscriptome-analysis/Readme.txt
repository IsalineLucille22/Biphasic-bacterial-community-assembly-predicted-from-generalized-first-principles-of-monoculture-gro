README

This folder has the following subfolders or data

1) Dataset_S4_genome_information.xlsx

Has the annotations and genome codes per soil isolate genome. Genome key codes needed for (5)

2) KO-group-plots

Scripts (.txt files) and results of the KEGG orthology and pathway mapping. 3) Lysobacter-DE

Script (.txt file) and results of the differential gene expression analysis for Lysobacter in the SynCom21-MT experiment exclusively for comparison of timepoint 38 vs 22 and 38 h vs 70 h. .xlsx file used as Dataset S14) PCA-analysis

PCA-analysis of the collective ko-annotations for either all strains or for the subset of eight strains in the SynCom21-MT experiment with sufficient coverage.5) SynCom21-combined-annotations.csv

Datafile with the combined KEGG annotations or other relevant database references, per genome of the 21 soil isolates and per annotated gene. Is used as Dataset S3.6) TPM-count-data

Individual normalized TPM gene expression data for each of the isolates in the SynCom21-MT experiment, either as log2-transformed or raw value.  Combined file of all log2-TPM transformed files used as Dataset S57) transporter-annotations

Results of transporter gene annotations and inferred transporter substrates. File '2026-etc.csv' is used as Dataset S2.8) transporters-expression-analysis

Scripts (.txt files) and results of the transporter-substrate expression changes in the SynCom21-MT experiment. 9) Uptake_groups.xlsx

Classification of transporter substrates as chemical groups.
